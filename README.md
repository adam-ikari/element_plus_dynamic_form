# 动态表单组件 (Dynamic Form)

本项目实现了一个基于 Vue 3 和 Element Plus 的动态表单组件，支持多种输入类型和条件显示功能。

## 演示功能

- 多种输入类型支持（文本框、选择框、日期选择器、开关、滑块等）
- 条件显示（根据下拉菜单选择显示不同字段）
- 表单验证
- 响应式设计
- 多国语支持（基于 vue-i18n 库实现）
